# 🟦 서비스 개요

> 웹캠을 통해 **중증 시각장애인의 요리 과정을 인식**하고, **재료의 상태와 위험 요소를 판단해 음성 피드백을 제공**하는 **요리 보조 서비스**입니다.
>
> 또한 사용자의 **음성을 인식**하여 **STT·TTS 기반의 양방향 음성 인터페이스를 제공**합니다.

## 🔵 기능 소개

- 시각장애인을 위한 맞춤형 레시피를 저장하고 음성 기반으로 조회할 수 있습니다.
- 레시피를 선택하면 요리가 시작되며, 웹캠을 통해 사용자의 요리 과정을 실시간으로 인식합니다.
- 재료의 익힘 정도, 현재 작업 위치, 잠재적인 위험 요소를 분석해 음성으로 안내합니다.
- 요리 진행 상황을 자동으로 인식해 다음 단계로 자연스럽게 이동합니다.
- 사용자의 음성 질문을 인식(STT)하고, 적절한 답변을 생성해 음성(TTS)으로 제공합니다.

## 🔵 역할 및 기여도

**Backend 개발 및 기획**

- 서비스 전반의 서버 아키텍처 및 핵심 로직을 설계·구현
- DB 설계 및 데이터 모델링
- 프론트엔드와 REST API 및 WebSocket 기반 실시간 음성 통신 구현
- 비전 서버와의 API 연동을 통해 재료 상태를 지속적으로 수신하고, 이를 기반으로 레시피 단계 추적 및 사용자 응답 생성
- 요리 진행 흐름과 음성 피드백 로직을 포함한 서버 전체 기능 구현
- AWS EC2 환경에 서버 배포 및 운영
- GitHub Actions를 활용한 CI/CD 파이프라인 구축으로 자동 빌드·배포 환경 구성
